# visuals
React component library for building visualizations

## install

```sh
$ npm install visuals
```

```js
const { Grid } = require('visuals');

<Grid />
```

## contributing

```sh
$ git clone https://github.com/joekent-cms-project/visuals.git
$ npm install
...
$ npm run storybook
$ npm run test
$ npm run lint
```
